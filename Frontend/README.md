# 🦷 Oral Cancer Detection & Management Platform

## 🎓 Graduation Project

A comprehensive web-based platform developed to support the **early detection of oral cancer**, improve doctor–patient communication, and provide a structured digital healthcare experience.

---

## 📌 Project Overview

Oral cancer is a serious and potentially life-threatening disease that requires early diagnosis and continuous monitoring.  

This project presents a modern healthcare web application designed to:

- Assist in preliminary oral cancer risk assessment  
- Connect patients with doctors efficiently  
- Simplify appointment booking and follow-up  
- Increase public awareness about oral cancer  

The system serves both **patients and medical professionals** through dedicated portals.

---

## 🚀 Main Features

### 👨‍⚕️ Doctor Portal
- Dashboard with patient overview  
- Review uploaded oral scans  
- Appointment management  
- Real-time messaging system  
- Patient activity tracking  

### 🧑 Patient Portal
- Upload oral images for quick risk assessment  
- Book doctors and hospitals  
- View medical reports and scan history  
- Track personal health progress  
- Access educational resources  

### ⚡ Quick Risk Assessment
- Image upload system  
- Preliminary risk scoring  
- Instant feedback  
- Follow-up recommendations  

---

## 🛠️ Technologies Used

- **Frontend:** React + TypeScript  
- **Styling:** Tailwind CSS  
- **Build Tool:** Vite  
- **State Management:** Context API  
- Modern UI/UX principles

## 🎯 Project Objectives

- Promote early detection of oral cancer  
- Increase awareness and education

## 👥 Target Users

- Patients concerned about oral health

## ⚠️ Disclaimer

This platform provides a **preliminary risk assessment only** and does not replace professional medical diagnosis. Users should consult qualified healthcare professionals for accurate medical evaluation.
